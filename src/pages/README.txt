This folder contains the main page components for the application.

Files:
- Home.tsx: General job postings and dashboard.
- Search.tsx: Job filtering and search interface.
- MyJobs.tsx: Saved and applied jobs tracking.
- Documents.tsx: Management of generated resumes and cover letters.
- Account.tsx: User profile and onboarding.
- Onboarding.tsx: Multi-step account setup flow.

Resources:
- React Router: https://reactrouter.com/
