This folder contains reusable UI components used across the application.

Files:
- Button.tsx: Standardized button component with variants.
- Card.tsx: Container for job listings and profile sections.
- Input.tsx: Styled input fields for forms.
- Navbar.tsx: Main navigation bar.
- Chatbot.tsx: Integrated AI career assistant.
- Modal.tsx: Reusable modal for document previews and confirmations.

Resources:
- Tailwind CSS: https://tailwindcss.com/docs
- Framer Motion: https://www.framer.com/motion/
